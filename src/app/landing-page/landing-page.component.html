Welcome to BookSavviebeta

BookSavvie is a book recommendation web application. When you signup to the application, you will be asked about genres that interest you along with the books you have already read. Then navigating to the home page shows you the recommendations.
We hope you'll love it as much as we loved developing it.

Rounick Palit

Full Stack Developer & Co-Product Designer

BookSavvie's core product planning and designing has been done by Mridul Dhar and me. From there on, with occasional guidance from my friends, I took charge and developed the entire web application end to end from scratch. Technologies used consist of ASP.Net Web API for the backend, MS SQL Server for the database, and HTML, CSS, Bootstrap and Angular 7 for the Frontend.
A big thanks to Samridh James and Shubham Srivastava for their support.
